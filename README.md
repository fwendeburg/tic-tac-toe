# tic-tac-toe
tic-tac-toe game.

## Features
- Can be played human vs. human, bot vs. bot or human vs. bot.
- Responsive design.

## Tech Stack
HTML, CSS, JavaScript

## Run Locally
Clone the project

```bash
  git clone https://github.com/fwendeburg/tic-tac-toe.git
```

Go to the project directory

```bash
  cd tic-tac-toe
```

Open the index.html file with any browser
